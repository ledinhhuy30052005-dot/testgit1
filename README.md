# Project Name

## Introduction
This project demonstrates how to use STM32 to control LED and read button input.

## Features
- Control LED using GPIO
- Read button input
- Send data through UART
- Simple embedded system example

## Hardware
- STM32F103C8T6
- ST-Link V2
- LED
- Push Button

## Software
- Keil uVision
- Visual Studio Code
- Git
